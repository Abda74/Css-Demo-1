# Projet HTML/CSS - Dégradés sur une Carte

Ce projet est une démonstration de l'utilisation créative des dégradés dans la conception d'une interface HTML/CSS. L'objectif principal était de créer une interface attrayante en utilisant des dégradés pour améliorer l'esthétique visuelle de la carte.

## Fonctionnalités

- **Dégradés Artistiques :** Les dégradés sont utilisés de manière artistique pour donner une apparence unique à la carte, créant une expérience visuelle agréable pour les utilisateurs.

- **Contenu Dynamique :** La carte comprend des informations dynamiques, telles que des emplacements spécifiques ou des zones d'intérêt, intégrées de manière interactive.

- **Styles Élégants :** Les styles CSS sont soigneusement choisis pour compléter les dégradés, créant une harmonie visuelle entre la carte et les textes.

## Comment Utiliser

1. Clonez le repository sur votre machine locale.
   ```bash
   git clone https://github.com/votre-nom/Projet-Degradé-Carte.git
   ```

2. Ouvrez le fichier `index.html` dans votre navigateur préféré.

3. Explorez la carte interactive et admirez les dégradés utilisés dans le projet.
